Snake in Java

Political and Societal Influence Project Explanation Essay
Main Premise. I decided to make this project to recreate the popular game "Snake", but with my own special twist! In "Blocky Snake", the more apples you eat the more blocks appear on the screen! Not only that, but you have to avoid the rotten apples; otherwise, you'll points and your snake will get shorter. Finally, all original snake mechanics are added.
New Mechanic: JFrame. One of the rquirements for this final project is that we must implement something that we haven't learned in class. A big part of my project fulfills this condition, which is JFrame. JFrame is a java swing class(?) that you must implement into your class. Once you create a "JFrame", a seperate window opens. In this window you can add graphics which you wouldn't be able to do without! This is very useful for projects like this as it allows you to use "Graphics" objects and create visuals for a game.
